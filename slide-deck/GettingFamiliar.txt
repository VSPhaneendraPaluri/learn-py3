Python & It's Eco-System
------------------------

This reading material is an effort to know the necessary eco-system created around Python

PSF :
  - Python Software Foundation is the organization driving Python Language Group and it's development.


PIP :
  - It is a package management system for managing packages written in Python

  - The latest being PIP3

  - The way to install PIP packages (modules) for one's application needs is to type

   >$ pip install <package-name>


PEPn (Python Enhancement Proposals):

  - n is the proposal index
